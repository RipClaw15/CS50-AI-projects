# CS50 Artificial Intelligence Portfolio

> *A comprehensive implementation of Harvard's CS50 AI course, refactored for clarity and packaged as a showcase of fundamental AI algorithms.*

## Why This Repository Exists

This isn't just homework. This is my practical foundation in classical AI and machine learning. Each project represents a core paradigm I can implement from scratch, explain in an interview, and adapt for real-world problems.

## Project Index

| Project | Core Concept | Tech Stack | Quick Take |
|---------|--------------|------------|------------|
| **[Traffic](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week5/traffic)** | **[Computer Vision (CNN)](https://en.wikipedia.org/wiki/Convolutional_neural_network)** | Python, TensorFlow (Keras), OpenCV, scikit-learn | Classifies road signs using a convolutional neural network, with OpenCV handling image preprocessing |
| **[Attention](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week6/attention)** | **[Transformer Attention Visualization](https://huggingface.co/docs/transformers/index)** | Python, TensorFlow, Hugging Face Transformers | Loads BERT to predict masked tokens and **generates self-attention heatmap diagrams from scratch**—visualizing how the model weighs relationships between words |
| **[Shopping](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week4/shopping)** | **[Supervised Learning (k-NN)](https://en.wikipedia.org/wiki/K-nearest_neighbors_algorithm)** | Python, scikit-learn | Predicts e-commerce purchase intent using a **k-Nearest Neighbors (k=1) classifier** with manual feature engineering for categorical data—and calculates sensitivity/specificity from scratch to evaluate model performance |
| **[Parser](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week6/parser)** | **[Natural Language Processing](https://www.ibm.com/think/topics/natural-language-processing)** | Python, NLTK | Context-free grammar for extracting noun phrases |
| **[Heredity](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week2/heredity)** | **[Bayesian Inference (Exhaustive Enumeration)](https://arxiv.org/pdf/1303.5714)** | Python | Brute-force calculates posterior genetic probabilities across all possible family trait combinations |
| **[Crossword](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week3/crossword)** | **[Constraint Satisfaction (CSP)](https://en.wikipedia.org/wiki/Constraint_satisfaction_problem)** | Python, Pillow | Solves crossword puzzles using **backtracking search enhanced with AC-3 arc consistency, MRV (minimum remaining values), and LCV (least constraining value) heuristics**—and renders the completed grid as an image ||
| **[Nim](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week4/nim)** | **[Reinforcement Learning (Q-Learning)](https://en.wikipedia.org/wiki/Q-learning)** | Python | Implements **tabular Q-learning from scratch** with epsilon-greedy exploration, Bellman equation updates, and sparse reward signals (`+1`/`-1`/`0`) to master the game of Nim—**no external ML libraries used** |
| **[Degrees](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week0/degrees)** | **[Shortest Path (BFS)](https://en.wikipedia.org/wiki/Breadth-first_search)** | Python | Finds the "Kevin Bacon" connection between actors using IMDb data |
| **[Tic-Tac-Toe](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week0/tictactoe)** | **[Adversarial Search (Minimax)](https://en.wikibooks.org/wiki/Artificial_Intelligence/Search/Adversarial_search/Minimax_Search)** | Python, Pygame | Undefeatable AI using brute-force Minimax search with a complete game tree exploration and a Pygame interface |
| **[Minesweeper](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week1/minesweeper)** | **[Knowledge Representation](https://en.wikipedia.org/wiki/Knowledge_representation_and_reasoning)** & **[Constraint Propagation](https://en.wikipedia.org/wiki/Constraint_satisfaction)** | Python, Pygame | Applies **subset inference and constraint propagation** to deduce safe moves and identify mines, with a playable graphical interface |
| **[Knights](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week1/knights)** | **[Propositional Logic](https://en.wikipedia.org/wiki/Propositional_logic)** & **[Model Checking](https://en.wikipedia.org/wiki/Model_checking)** | Python | Solves classic "Knights and Knaves" puzzles using a **custom-built model checker** that brute-forces all possible truth assignments to determine who lies and who tells the truth |
| **[PageRank](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week2/pagerank)** | **[Markov Chains](https://en.wikipedia.org/wiki/Markov_chain)** & **[Iterative Convergence](https://leancrew.com/all-this/2020/02/iteration-and-convergence/)** | Python | Implements the PageRank algorithm **using both Monte Carlo sampling (10,000 random surfer walks) and mathematical iterative convergence**—with proper handling of dangling pages—to rank web importance |

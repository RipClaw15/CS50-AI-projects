# CS50 Artificial Intelligence Portfolio

> *A comprehensive implementation of Harvard's CS50 AI course, refactored for clarity and packaged as a showcase of fundamental AI algorithms.*

## Why This Repository Exists

This isn't just homework. This is my practical foundation in classical AI and machine learning. Each project represents a core paradigm I can implement from scratch, explain in an interview, and adapt for real-world problems.

## Project Index

| Project | Core Concept | Tech Stack | Quick Take |

| **[Degrees]([link-to-folder](https://github.com/RipClaw15/CS50-AI-projects/tree/main/week0/degrees))** | Shortest Path (BFS) | Python | Finds the "Kevin Bacon" connection between actors using IMDb data |

| **[PageRank](link-to-folder)** | Random Surfer Model | Python | Maps web page importance via Markov chains |
